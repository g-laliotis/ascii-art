Installation & Usage

Quick Start

git clone https://github.com/g-laliotis/ascii-art.git cd ascii-art go run ./cmd/ascii-art "Hello World"

Build from Source

make build ./ascii-art "Your Text Here"

Using Makefile

make test # Run all tests make build # Build binary make install # Install to GOPATH/bin make run # Run with example
